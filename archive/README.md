# Archive Folder

This folder contains legacy or utility files that are no longer part of the main application runtime.

These files are preserved for historical reference, migration utilities, or one-off maintenance tasks.

## Included files

- `apply_indexes.py` — old database index helper
- `check_db.py` — legacy database health check
- `db_schema.txt` — schema notes
- `fix_db.py` — legacy database repair script
- `migrate_db.py` — legacy SQLite migration helper
- `migrate_payments.py` — legacy payment migration helper
- `nextstep_no_bg.png` — unused artwork asset
- `promote_admin.py` — legacy admin promotion helper
- `templates_usage.txt` — unused template documentation
- `update_db.py` — legacy database updater
- `update_db_v2.py` — legacy database updater
- `verify_phase2_logic.py` — legacy phase2 validation helper
- `voices.json` — unused voice asset

## Notes

- The main application runs from `app/`.
- Utility scripts remain in `scripts/`.
- If you need any archived file, it is safe to restore from this folder.
